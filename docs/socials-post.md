# Socials post — Agentic World of Legends

🚀🎮 **Shipped: Agentic World of Legends (beta)** — a browser platformer where
players progress through six GitHub credential realms:
GH-100, GH-900, GH-500, GH-300, GH-200, and GH-600.

Each level ends with a quiz gate before progression, then the player enters a
final Mainline showdown.

Built with Phaser 3 + TypeScript + Vite, tested with Vitest, and deployed with
GitHub Actions + Pages.

👉 Play: https://ghlearn.github.io/agentic-world-legends/
